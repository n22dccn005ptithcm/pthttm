# pthttm
Ứng dụng học tiếng Anh bằng Flash Card
1. Cài đặt Java (JRE hoặc JDK)
Máy bạn cần có Java Runtime Environment (JRE) hoặc Java Development Kit (JDK).
Khuyến nghị dùng phiên bản Java 8 trở lên.
Kiểm tra Java đã có chưa bằng lệnh:
java -version

2. Chạy file .jar
Sau khi cài Java, bạn có thể chạy bằng Terminal/CMD:
java -jar Feel_Gemini.jar
